# COMP7940 Project - CloudCampus AI Bot

## Summary
A cloud-deployed Telegram chatbot powered by an LLM API.  
The system supports intelligent question answering, message logging to a cloud database, containerized deployment, and CI/CD automation.

## Tech Stack
- Python
- Telegram Bot API
- FastAPI
- OpenAI API
- Supabase Postgres
- Docker
- AWS EC2
- GitHub Actions

## Features
- Telegram chatbot
- LLM-based responses
- Cloud database logging
- Health check endpoint
- Dockerized deployment
- CI/CD pipeline
